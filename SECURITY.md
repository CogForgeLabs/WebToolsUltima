# Security Policy

## Reporting a Vulnerability

Cognitive Industries / CogForgeLabs takes security seriously. If you believe you
have found a security vulnerability in this project, please report it responsibly.

**Do not report security vulnerabilities through public GitHub issues.**

Email: **security@cognitive-industries.org** (or contact@cognitive-industries.org)

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce (proof-of-concept if possible)
- Affected version(s) or commit hash
- Any suggested remediation

## What to expect
- We acknowledge receipt within 5 business days.
- We investigate and keep you informed of progress.
- We ask for a reasonable window to remediate before public disclosure.

## Secrets
Secrets, credentials, and environment files are never committed to this repository.
If you find any exposed secret in the history, please report it immediately.
