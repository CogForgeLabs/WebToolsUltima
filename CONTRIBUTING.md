# Contributing

Thanks for your interest in this project by Cognitive Industries / CogForgeLabs.
Contributions are welcome from everyone — humans and AI assistants alike. We
evaluate contributions on their merit: correct, safe, well-tested, and a good fit.

## Getting started
- Fork the repo (or branch from `main`) and open a pull request.
- Read the README for build and run instructions.

## Development
- Keep changes focused and well-described.
- **Never commit secrets, credentials, `.env` files, keys, or tokens.** Use `.env.example` where present.
- Match the existing code style of the surrounding files.

## Pull requests
- Describe what changed and why.
- Ensure the project builds and any tests pass.
